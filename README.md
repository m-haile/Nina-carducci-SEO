## 🎯 Overview

Nina Carducci is a professional photographer. This portfolio website has been optimized for:
- **Search Engine Visibility**: Implemented SEO best practices
- **Performance**: Fast load times and optimized assets
- **Accessibility**: WCAG compliance and screen reader support
- **User Experience**: Responsive design and intuitive navigation

## 🛠️ Tech Stack

- **HTML5** (52.1%)
- **JavaScript** (29.5%)
- **CSS3** (18.4%)

## ✨ Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **SEO Optimized**: Meta tags, structured data, and semantic HTML
- **Performance Optimized**: Optimized images, minified assets, and efficient code
- **Accessibility First**: WCAG 2.1 compliant with semantic markup
- **Rich Snippets**: Google rich snippet implementation for enhanced search results
- **Fast Loading**: Optimized for Core Web Vitals
